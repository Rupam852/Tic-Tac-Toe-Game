# Tic-Tac-Toe Live

This is a modern, full-featured Tic-Tac-Toe game with real-time multiplayer, robot matches, global leaderboards, sound effects, and secure accounts.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
